# splitPolygon — 切断線によるポリゴン分割

## この機能が解決する問題

ユーザが地図上でポリゴンを描いた後、「この道路で区画を分けたい」「この境界で2つのエリアに分割したい」
という場面が頻繁に発生する。

```
     切断線（ドラフト）
         ↓
  ──── ╲ ─────────────
  │     ╲      B     │
  │      ╲          │
  │  A    ╲         │     ← 1つのポリゴンを
  │        ╲        │        切断線で2つに分割
  │         ╲       │
  ──────────  ╲ ─────
                ╲
```

`splitPolygon` は切断線（開いたドラフト）を受け取り、ポリゴンを2つ以上のピースに分割する。

---

## 類似 API との違い

| | `splitPolygon` | `resolveOverlapsWithDraft` |
|---|---|---|
| **入力のドラフト** | ポリゴンを貫通する切断線 | ポリゴンを横切るドラフト線 |
| **結果** | 元ポリゴンが**削除**され、全ピースが新規ポリゴン | 元ポリゴンは**縮小**（ID維持）、交錯部のみ新規 |
| **用途** | 意図的な分割 | 描画中の交錯の自動処理 |

| | `splitPolygon` | `carveInnerPolygon` |
|---|---|---|
| **切断の方向** | **外から外**（ポリゴンを貫通） | **境界から内側**（ループ） |
| **結果の形状** | 2つ以上の独立ピース | C字型の外側 + 内側の2つ |

---

## ユースケースとアプリ側の呼び出しフロー

### 想定シーン

道路に沿ってポリゴンを二分したい。ユーザはポリゴンの外側から線を引き始め、
ポリゴンの中を通って反対側の外に出る。

### 呼び出し例

```typescript
// ユーザが切断線の描画を完了した時点で
const cutLine: DraftShape = {
  points: [
    { lat: -1, lng: 2 },  // 外側（ヒゲ — 自動除去される）
    { lat: 1, lng: 2 },   // ポリゴン辺との交点付近
    { lat: 3, lng: 2 },   // ポリゴン辺との交点付近
    { lat: 5, lng: 2 },   // 外側（ヒゲ — 自動除去される）
  ],
  isClosed: false,
};

const pieces = await editor.splitPolygon(targetPolygonId, cutLine);
// pieces = [ピース1(新規ID), ピース2(新規ID)]
// 元のポリゴンは削除されている
```

---

## API

```typescript
splitPolygon(
  polygonId: PolygonID,
  draft: DraftShape          // isClosed = false
): Promise<MapPolygon[]>
```

- 元のポリゴンは**削除**される
- 分割結果の各ポリゴンは新規 ID、`display_name` は空文字
- 分割が発生しなかった場合は空配列を返す

---

## 動作の図解

### ヒゲ（端部）の自動除去

ユーザがポリゴンの外側から線を引いた場合、境界の外にはみ出した部分（ヒゲ）は
ライブラリが自動的に除去する。ユーザは交差点を正確に狙う必要がない。

```
描画中:  外側 ─── I1 ─────────── I2 ─── 外側
              ↑ヒゲ    実際の切断線    ヒゲ↑
              （自動除去）           （自動除去）

確定後:           I1 ─────────── I2
```

### 交差数による挙動

| 交差数 | 挙動 | 図 |
|--------|------|----|
| 0 | 何もしない | — |
| 1 | 頂点挿入のみ（分割なし） | 辺上に新頂点が追加される |
| 2 | 1分割 → 2ピース | 下図参照 |
| 4 | 2分割 → 3ピース | 切断線がジグザグにポリゴンを横切る場合 |

### 2交差点での分割

```
入力:
  ┌──────────────┐         切断線: (-1,2) → (5,2)
  │              │
  │              │         交差点: I1=(0,2), I2=(4,2)
  │   元の       │
  │   ポリゴン    │
  │              │
  └──────────────┘

結果:
  ┌──────────────┐
  │   ピース 1    │ ← 新規 ID
  ├──────────────┤  I1 ────── I2  (切断線)
  │   ピース 2    │ ← 新規 ID
  └──────────────┘
  （元のポリゴンは削除）
```

---

## エッジケースと挙動

| ケース | 挙動 |
|--------|------|
| 交差なし | 空配列を返す。ポリゴン変更なし |
| 交差1点 | 辺上に頂点を挿入するのみ。分割なし |
| 奇数交差（2N+1） | N回分割 + 1頂点挿入 |
| 切断線がポリゴンの頂点を通る | 既存頂点がある場合は重複挿入しない |

---

## 制約・注意事項

| 項目 | 内容 |
|------|------|
| **元ポリゴン** | 削除される（ID は消滅） |
| **結果ポリゴン** | 全て新規 ID、display_name は空文字 |
| **Undo** | 元ポリゴンを復元し、分割ピースを削除 |
| **ドラフトの状態** | `isClosed = false` であること |
